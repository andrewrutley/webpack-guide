# A Beginner’s Guide to Webpack 4

```
mkdir webpack-guide && cd webpack-guide
npm init -y
npm install --save-dev webpack webpack-cli
```

Next we’ll create the following directory structure and contents:

```
  webpack-guide
  |- package.json
+ |- webpack.config.js
+ |- /src
+   |- app.js
```
